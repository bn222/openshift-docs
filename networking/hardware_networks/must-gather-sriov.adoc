= Collecting must-gather for the SR-IOV Network Operator
include::_attributes/common-attributes.adoc[]
:context: about-sriov

toc::[]

To collect the must-gather information for pods ran by the SR-IOV Network Operator, the standard `oc adm` command can be used by specifying the mustgather image for the sriov network operator.

You can enable SR-IOV on a node by using the following command:
[source,terminal]
----
$ oc adm must-gather --image registry.redhat.io/openshift4/ose-sriov-operator-must-gather
----
